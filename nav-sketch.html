Side-by-Side Comparison
Option A — 5 + More ▼  Recommended

Recommendation — Option A with smart grouping
Keep 5 primary tabs always visible: Dashboard  •  Plans  •  Map  •  Calendar  •  Requests

Move everything else into a More ▼ dropdown with labeled sections:
  • Analysis → Timeline, Reports
  • Compliance → Compliance, Library, Reference
  • Admin → System Log, Team

When you are inside an overflow view the button label shows the active view name (e.g. Reports ▼), so you never lose your position. This is the smallest change with the biggest visual impact — tab bar goes from 12 items down to 6 (5 tabs + 1 button).
